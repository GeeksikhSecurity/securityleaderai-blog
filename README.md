# Security Leader AI Blog

Professional blog for SecurityLeader.ai featuring AI security, MCP research, and cybersecurity leadership content.

## Features

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Markdown blog posts
- SEO optimized
- Responsive design

## Blog Posts

1. **MCP Sentinel Scanner** - Research-backed security analysis tool for Model Context Protocol
2. **Security ROI** - Why credential management delivers better ROI than APT defense
3. **DevOps Security Paradox** - When more tools mean less security

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deploys automatically to Vercel on push to main branch.

Domain: securityleader.ai

## Adding Posts

Create markdown files in `/posts` directory with frontmatter:

```md
---
title: "Your Title"
date: "2025-10-12"
excerpt: "Brief description"
author: "Author Name"
tags: ["tag1", "tag2"]
---

Your content here...
```
